Framework Dimension Definitions

  • Open Input Data: Identifiable, documented, attributable, versioned raw measurement evidence inputs.
  • Open-Source Software: Inspectability of Analytical logic via code repositories, configurations, and license files.
  • Open Execution: Computational environments, workflows, run logs, and outputs make rerun processes inspectable.
  • Open Community & Review: Scrutiny, peer review, issue tracking, and responses to replication challenges are visible.
  • Open Publication: Resulting reports, papers, and dashboards are openly accessible and reference source materials.
  • Open Linkage: The versioned, explicit connectivity linking data, models, execution, review, and publication steps.
